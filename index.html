Cheatsheet

Typography

The art of arranging text on a page.

Property Usage Description
font-family font-family: Georgia, Helvetica, serif; Font family or typeface is the type of font. Can contain fallback fonts along with the primary font.
font-weight font-weight: [bold | normal | 100-900]; The width of the text. Can be bold/normal or a number value between 100 and 900.
font-style font-style: italic; Italices the text.
word-spacing word-spacing: 0.3em; Spacing between words in a body of text. Result will be normal spacing plus entered value.
letter-spacing letter-spacing: 0.2em; Spacing between letters in a text.
text-transform text-transform: [uppercase | lowercase]; Transforms the selected text to upper or lower case.
text-align text-align: [left | right| center]; Horizontally aligns the selected text left, right or center.
